Tutorial Completo del Sistema de Gestión de Bots de Trading

Introducción

Este sistema permite gestionar bots de trading en múltiples exchanges de criptomonedas. Puedes configurar diferentes exchanges, añadir pares de trading (tokens), y controlar las estrategias de trading para cada par.

Página Principal: Estado de los Exchanges

En la página principal, encontrarás una tabla con todos los exchanges configurados. Cada fila muestra:

  • ID del Exchange: Identificador único para cada exchange en el sistema.
  • Nombre del Exchange: El nombre del exchange (ej. Binance, OKX).
  • Estado: Indica si el bot está actualmente operando en este exchange.
  • Acciones:
    • Iniciar/Detener: Controla la actividad del bot en este exchange.
    • Editar: Modifica la configuración del exchange.
    • Eliminar: Elimina el exchange del sistema.
    • Ver Tokens: Muestra los pares de trading configurados para este exchange.

Agregar un Nuevo Exchange

Para comenzar a operar en un nuevo exchange, primero debes agregarlo al sistema:

  1. Haz clic en "Agregar Exchange" en la página principal.
  2. Completa los siguientes campos:
    • ID del Exchange: Un identificador único que usarás para referirte a este exchange en el sistema.
    • Nombre del Exchange: El nombre oficial del exchange (ej. okx, binance). Debe coincidir exactamente con el nombre reconocido por el sistema.
    • API Key: La clave API de tu cuenta en el exchange. Se usa para autenticar las solicitudes del bot.
    • Secret: La clave secreta asociada con tu API Key. Nunca compartas esta información.
    • Password: Algunos exchanges requieren una contraseña adicional. Si no es necesario, déjalo en blanco.
  3. Haz clic en "Agregar Exchange" para guardar la configuración.

Nota: Asegúrate de que las claves API tengan los permisos necesarios para realizar operaciones de trading.

Editar la Configuración de un Exchange

Si necesitas actualizar las credenciales o cambiar la configuración de un exchange:

  1. Localiza el exchange en la página principal y haz clic en "Editar".
  2. Actualiza los campos necesarios:
    • API Key: Si has generado una nueva clave API.
    • Secret: La nueva clave secreta asociada.
    • Password: Si el exchange lo requiere y ha cambiado.
  3. Haz clic en "Guardar Cambios" para actualizar la información.

Importante: Cambiar estas credenciales afectará inmediatamente a las operaciones del bot en este exchange.

Gestión de Tokens

Agregar Tokens al Exchange

Los tokens representan los pares de trading que el bot gestionará en cada exchange.

  1. En la página principal, haz clic en "Ver Tokens" junto al exchange deseado.
  2. En la página de gestión de tokens, haz clic en "Agregar Token".
  3. Completa los detalles del token:
    • Símbolo: El par de trading (ej. BTC/USDT). Debe coincidir exactamente con el formato del exchange.
    • Spread: El porcentaje de diferencia entre el precio de compra y venta. Determina el margen de ganancia mínimo.
    • Take Profit: El porcentaje de ganancia objetivo antes de vender.
    • Cantidad de Trade: El monto a operar en cada transacción.
    • Número Máximo de Órdenes: Limita la cantidad de órdenes abiertas simultáneas.
    • Tiempo de Expiración de Órdenes: Tiempo en minutos antes de que una orden pendiente sea cancelada.
    • Máximo de Pérdida Diaria: Límite de pérdida que, al alcanzarse, detiene las operaciones del token por ese día.
    • Máximo de Órdenes de Venta: Limita la cantidad de órdenes de venta abiertas.
    • Umbral de Reactivación: Porcentaje de recuperación necesario para reactivar el trading después de alcanzar la pérdida máxima diaria.
  4. Haz clic en "Agregar Token" para finalizar.

Editar un Token Existente

Para ajustar la estrategia de trading de un token:

  1. En la lista de tokens del exchange, haz clic en "Editar" junto al token deseado.
  2. Modifica los parámetros según sea necesario.
  3. Haz clic en "Guardar Cambios" para aplicar las modificaciones.

Eliminar un Token

Si ya no deseas operar un par específico:

  1. Encuentra el token en la lista del exchange.
  2. Haz clic en "Eliminar" junto al token.
  3. Confirma la acción cuando se te solicite.

Nota: Eliminar un token detendrá todas las operaciones relacionadas y cancelará las órdenes abiertas.

Control del Bot

Iniciar el Bot: Para comenzar las operaciones en un exchange, haz clic en "Iniciar" junto al exchange en la página principal. Esto activará el bot para todos los tokens configurados en ese exchange.

Detener el Bot: Para pausar todas las operaciones, haz clic en "Detener". Esto cancelará todas las órdenes pendientes y detendrá nuevas operaciones.

Importante: Iniciar o detener el bot afecta a todos los tokens configurados en ese exchange.

Monitoreo del Bot

El sistema proporciona varias formas de seguir el rendimiento de tus operaciones:

  • Órdenes Abiertas: Visualiza todas las órdenes actualmente en el mercado.
  • Ganancias/Pérdidas (PnL): Revisa el rendimiento financiero de cada token y del exchange en general.
  • Precios Actuales: Observa los precios en tiempo real de los tokens que estás operando.

Usa estas herramientas para tomar decisiones informadas sobre tus estrategias de trading.

Consideraciones de Seguridad

  • Nunca compartas tus claves API o contraseñas con nadie.
  • Utiliza claves API con permisos limitados cuando sea posible.
  • Monitorea regularmente la actividad de tu cuenta para detectar cualquier anomalía.
  • Considera usar autenticación de dos factores en tus cuentas de exchange.
